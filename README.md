# Política de Privacidad de TetApp

Última actualización: 2026-06-25

TetApp es una aplicación para registrar lactancia, peso, notas y recordatorios familiares. Esta política explica qué datos se tratan, para qué se usan y cómo podés solicitar su eliminación.

Antes de publicarla, reemplazar estos datos:

- Responsable: `Demaría Sebastián Federico`
- Email de contacto: `sebastianf.demaria@gmail.com`
- Url politica de privacidad: `sebdemaria.github.io`

## Datos Que Registrás En La App

TetApp permite guardar la siguiente información en una sala compartida:

- Código de sala.
- Registros de lactancia, incluyendo lado, fecha, hora, duración y notas opcionales.
- Registros de peso del bebé.
- Configuración de alertas y recordatorios.
- Preferencias visuales de la sala.
- Miembros/dispositivos con acceso a la sala.

No solicitamos nombre, email, teléfono ni una cuenta personal tradicional dentro de la app. El acceso a una sala se realiza mediante código de sala y contraseña.

## Datos Técnicos

Para que la app funcione correctamente, se pueden tratar datos técnicos como:

- Identificador técnico de instalación generado por la app.
- Identificador anónimo usado para validar acceso con Firebase Authentication.
- Información técnica necesaria para sincronizar datos con Firebase.

Estos datos se usan para autenticar el dispositivo, mantener la sesión de sala y sincronizar la información entre dispositivos autorizados.

## Finalidad Del Tratamiento

Usamos los datos para:

- Registrar y mostrar información de lactancia y peso.
- Sincronizar datos entre dispositivos que pertenecen a la misma sala.
- Programar recordatorios locales de lactancia.
- Permitir exportar registros a un archivo Excel.
- Proteger el acceso a cada sala mediante contraseña y reglas de seguridad.

No vendemos datos personales. No usamos los datos para publicidad personalizada.

## Servicios De Terceros

TetApp usa Firebase de Google para autenticación, base de datos y funciones backend:

- Firebase Authentication.
- Cloud Firestore.
- Cloud Functions for Firebase.

Firebase puede procesar datos técnicos necesarios para operar estos servicios. Podés consultar la documentación de privacidad de Google/Firebase en https://firebase.google.com/support/privacy.

La app también puede abrir servicios externos como Spotify si el usuario toca el acceso rápido correspondiente. TetApp no controla las prácticas de privacidad de esos servicios externos.

## Permisos Del Dispositivo

TetApp puede solicitar permisos del dispositivo solo cuando son necesarios:

- Cámara: para abrir la cámara desde accesos rápidos si el usuario decide hacerlo.
- Fotos/Galería: para guardar fotos tomadas desde la cámara, si el sistema lo requiere.
- Notificaciones: para recordatorios de lactancia y alertas configuradas por el usuario.
- Vibración/Sonido: para mejorar alertas y feedback de interacción.

Podés administrar estos permisos desde la configuración del sistema operativo.

## Datos De Salud Y Aviso Médico

TetApp no reemplaza consejo, diagnóstico ni tratamiento médico profesional. La información registrada es de uso organizativo familiar. Ante dudas sobre lactancia, peso, alimentación o salud del bebé, consultá a un profesional de salud.

## Compartir Datos

Los datos de una sala son visibles para los dispositivos que ingresen con el código de sala y la contraseña correcta. Compartir esos datos con otros dispositivos o personas depende del usuario que comparte las credenciales de la sala.

No compartimos datos con anunciantes ni brokers de datos.

## Conservación Y Eliminación

Los datos se conservan mientras exista la sala en Firebase.

Desde la app se puede eliminar una sala y sus datos desde Ajustes, usando la contraseña de la sala. Esta acción elimina registros, pesos, ajustes y accesos asociados a esa sala.

También podés solicitar asistencia para eliminar datos escribiendo a `sebastianf.demaria@gmail.com` e indicando el código de sala, siempre que podamos verificar razonablemente la solicitud.

## Seguridad

TetApp protege el acceso a salas mediante contraseña y reglas de seguridad de Firebase. Las contraseñas se almacenan como hashes en el backend, no en texto plano.

Aun así, ningún sistema es completamente infalible. Recomendamos usar contraseñas difíciles de adivinar y no compartir el código de sala con personas no autorizadas.

## Menores

TetApp está pensada para uso por adultos responsables del cuidado del bebé. La app no está dirigida a niños ni busca recopilar datos directamente de menores.

## Cambios En Esta Política

Podemos actualizar esta política para reflejar cambios en la app, requisitos legales o servicios utilizados. La fecha de actualización indicará la versión vigente.

## Contacto

Para consultas de privacidad o eliminación de datos, escribí a:

`sebastianf.demaria@gmail.com`

# TetApp Privacy Policy

Last updated: 2026-06-25

TetApp is an application for tracking breastfeeding, weight, notes, and family reminders. This policy explains what data is processed, what it is used for, and how you can request its deletion.

Before publishing this policy, replace the following information:

* Controller: `Demaría Sebastián Federico`
* Contact email: `sebastianf.demaria@gmail.com`
* Privacy policy URL: `sebdemaria.github.io`

## Data You Register In The App

TetApp allows you to save the following information in a shared room:

* Room code.
* Breastfeeding records, including side, date, time, duration, and optional notes.
* Baby weight records.
* Alert and reminder settings.
* Room visual preferences.
* Members/devices with access to the room.

We do not request your name, email address, phone number, or a traditional personal account within the app. Access to a room is done through a room code and password.

## Technical Data

In order for the app to work properly, technical data may be processed, such as:

* Technical installation identifier generated by the app.
* Anonymous identifier used to validate access with Firebase Authentication.
* Technical information required to synchronize data with Firebase.

This data is used to authenticate the device, maintain the room session, and synchronize information between authorized devices.

## Purpose Of Processing

We use the data to:

* Record and display breastfeeding and weight information.
* Synchronize data between devices that belong to the same room.
* Schedule local breastfeeding reminders.
* Allow records to be exported to an Excel file.
* Protect access to each room through a password and security rules.

We do not sell personal data. We do not use data for personalized advertising.

## Third-Party Services

TetApp uses Google Firebase for authentication, database, and backend functions:

* Firebase Authentication.
* Cloud Firestore.
* Cloud Functions for Firebase.

Firebase may process technical data necessary to operate these services. You can review Google/Firebase privacy documentation at https://firebase.google.com/support/privacy.

The app may also open external services such as Spotify if the user taps the corresponding quick access shortcut. TetApp does not control the privacy practices of those external services.

## Device Permissions

TetApp may request device permissions only when necessary:

* Camera: to open the camera from quick access shortcuts if the user decides to do so.
* Photos/Gallery: to save photos taken with the camera, if required by the system.
* Notifications: for breastfeeding reminders and alerts configured by the user.
* Vibration/Sound: to improve alerts and interaction feedback.

You can manage these permissions from your operating system settings.

## Health Data And Medical Disclaimer

TetApp does not replace professional medical advice, diagnosis, or treatment. The information recorded is intended for family organization purposes. If you have questions about breastfeeding, weight, feeding, or your baby’s health, please consult a healthcare professional.

## Data Sharing

Data from a room is visible to devices that enter with the correct room code and password. Sharing that data with other devices or people depends on the user who shares the room credentials.

We do not share data with advertisers or data brokers.

## Retention And Deletion

Data is retained as long as the room exists in Firebase.

From the app, you can delete a room and its data from Settings by using the room password. This action deletes records, weight entries, settings, and access associated with that room.

You can also request assistance to delete data by writing to `sebastianf.demaria@gmail.com` and indicating the room code, provided that we can reasonably verify the request.

## Security

TetApp protects access to rooms through passwords and Firebase security rules. Passwords are stored as hashes in the backend, not in plain text.

However, no system is completely foolproof. We recommend using passwords that are difficult to guess and not sharing the room code with unauthorized people.

## Minors

TetApp is intended to be used by adults responsible for caring for the baby. The app is not directed at children and does not seek to collect data directly from minors.

## Changes To This Policy

We may update this policy to reflect changes in the app, legal requirements, or services used. The update date will indicate the current version.

## Contact

For privacy questions or data deletion requests, please write to:

`sebastianf.demaria@gmail.com`
