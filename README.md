# Política de Privacidad de TetApp

Última actualización: 2026-06-25

TetApp es una aplicación para registrar lactancia, peso, notas y recordatorios familiares. Esta política explica qué datos se tratan, para qué se usan y cómo podés solicitar su eliminación.

Antes de publicarla, reemplazar estos datos:

- Responsable: `Demaría Sebastián Federico`
- Email de contacto: `sebastianf.demaria@gmail.com`
- Url politica de privacidad: `sebdemaria.github.io`

## Datos Que Registrás En La App

TetApp permite guardar la siguiente información en una sala compartida:

- Código de sala.
- Registros de lactancia, incluyendo lado, fecha, hora, duración y notas opcionales.
- Registros de peso del bebé.
- Configuración de alertas y recordatorios.
- Preferencias visuales de la sala.
- Miembros/dispositivos con acceso a la sala.

No solicitamos nombre, email, teléfono ni una cuenta personal tradicional dentro de la app. El acceso a una sala se realiza mediante código de sala y contraseña.

## Datos Técnicos

Para que la app funcione correctamente, se pueden tratar datos técnicos como:

- Identificador técnico de instalación generado por la app.
- Identificador anónimo usado para validar acceso con Firebase Authentication.
- Información técnica necesaria para sincronizar datos con Firebase.

Estos datos se usan para autenticar el dispositivo, mantener la sesión de sala y sincronizar la información entre dispositivos autorizados.

## Finalidad Del Tratamiento

Usamos los datos para:

- Registrar y mostrar información de lactancia y peso.
- Sincronizar datos entre dispositivos que pertenecen a la misma sala.
- Programar recordatorios locales de lactancia.
- Permitir exportar registros a un archivo Excel.
- Proteger el acceso a cada sala mediante contraseña y reglas de seguridad.

No vendemos datos personales. No usamos los datos para publicidad personalizada.

## Servicios De Terceros

TetApp usa Firebase de Google para autenticación, base de datos y funciones backend:

- Firebase Authentication.
- Cloud Firestore.
- Cloud Functions for Firebase.

Firebase puede procesar datos técnicos necesarios para operar estos servicios. Podés consultar la documentación de privacidad de Google/Firebase en https://firebase.google.com/support/privacy.

La app también puede abrir servicios externos como Spotify si el usuario toca el acceso rápido correspondiente. TetApp no controla las prácticas de privacidad de esos servicios externos.

## Permisos Del Dispositivo

TetApp puede solicitar permisos del dispositivo solo cuando son necesarios:

- Cámara: para abrir la cámara desde accesos rápidos si el usuario decide hacerlo.
- Fotos/Galería: para guardar fotos tomadas desde la cámara, si el sistema lo requiere.
- Notificaciones: para recordatorios de lactancia y alertas configuradas por el usuario.
- Vibración/Sonido: para mejorar alertas y feedback de interacción.

Podés administrar estos permisos desde la configuración del sistema operativo.

## Datos De Salud Y Aviso Médico

TetApp no reemplaza consejo, diagnóstico ni tratamiento médico profesional. La información registrada es de uso organizativo familiar. Ante dudas sobre lactancia, peso, alimentación o salud del bebé, consultá a un profesional de salud.

## Compartir Datos

Los datos de una sala son visibles para los dispositivos que ingresen con el código de sala y la contraseña correcta. Compartir esos datos con otros dispositivos o personas depende del usuario que comparte las credenciales de la sala.

No compartimos datos con anunciantes ni brokers de datos.

## Conservación Y Eliminación

Los datos se conservan mientras exista la sala en Firebase.

Desde la app se puede eliminar una sala y sus datos desde Ajustes, usando la contraseña de la sala. Esta acción elimina registros, pesos, ajustes y accesos asociados a esa sala.

También podés solicitar asistencia para eliminar datos escribiendo a `sebastianf.demaria@gmail.com` e indicando el código de sala, siempre que podamos verificar razonablemente la solicitud.

## Seguridad

TetApp protege el acceso a salas mediante contraseña y reglas de seguridad de Firebase. Las contraseñas se almacenan como hashes en el backend, no en texto plano.

Aun así, ningún sistema es completamente infalible. Recomendamos usar contraseñas difíciles de adivinar y no compartir el código de sala con personas no autorizadas.

## Menores

TetApp está pensada para uso por adultos responsables del cuidado del bebé. La app no está dirigida a niños ni busca recopilar datos directamente de menores.

## Cambios En Esta Política

Podemos actualizar esta política para reflejar cambios en la app, requisitos legales o servicios utilizados. La fecha de actualización indicará la versión vigente.
